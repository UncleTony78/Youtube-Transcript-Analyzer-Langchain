# YouTube Transcript Analysis System

## Current Phase: Initial Development - Transcript Retrieval

### Project Overview
A system to retrieve, analyze, and store YouTube video transcripts and metadata, with a credit-based access system.

### Credit System
- Free tier: 50 credits per day
- Actions and Credit Costs:
  - Video analysis (transcript + metadata): 10 credits
  - Text analysis: 5 credits
  - (More actions to be defined)
